2. Reproduce The bug
3. Play Computer
4. Fix The Errors
5. Print is your friend
6. Use a debugger (pythontutor.com)
7. Take a break
8. Ask a friend
9. Run Often (after you to a little execution check if it is working)
10. Ask Stack Overflow
